# PDMPSamplers.jl

**PDMPSamplers.jl** is in-development research software for Bayesian inference using Piecewise Deterministic Markov Processes (PDMPs).

## Status

:warning: **This package is under active development and not yet registered. Expect breaking changes and limited support.**

## Installation

Since `PDMPSamplers.jl` is not registered, install it directly from the repository:

```julia
using Pkg
Pkg.add(url="https://github.com/vandenman/PDMPSamplers.jl")
```

or type `]` in the Julia REPL and run

```julia
pkg> add https://github.com/vandenman/PDMPSamplers.jl
```
